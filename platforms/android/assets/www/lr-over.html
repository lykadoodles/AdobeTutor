Adobe Lightroom

Lightroom Module Overview

  Now that you know what Modules are, it’s time to talk about each Module individually. Adobe initially started with a set of core tools in Lightroom. Over years, it has been increasing the number of modules and enhancing their functionality with each new version. For example, the first three versions of Lightroom had five Modules – Library, Develop, Slideshow, Print and Web. Lightroom 4 and 5 added two additional Modules, Map and Book. Each module went through a number of rather significant changes as well – existing tools have been enhanced and new tools have been introduced. Let’s talk about each Module in more detail.
